# Mint

<p align="center">
  <img src="https://github.com/bewf/Mint/raw/main/GitAssets/minticon.png" width="220">
</p>

**Mint** is a client-side Bedwars resource tracker for Minecraft.

It tracks how many resources you have in your inventory and in your Ender Chest. 

I made alternatives have a _meh_ HUD, and don't support hypixel's new punch to deposit feature.

If you run into issues or have suggestions, feel free to open an issue on [GitHub](https://github.com/bewf/Mint/issues).


## Features

- **Live resource tracking**
  - Iron, Gold, Diamond, and Emerald
  - Tracks both inventory and Ender Chest totals.
- **Supports all deposit methods**
  - Manual Ender Chest deposits
  - Hypixelâ€™s punch-to-deposit mechanic.
- **OneConfig HUD**
  - Drag, scale, and toggle each resource.
  - Fully integrated with the OneConfig HUD editor.
- **Resource-based icons**
  - Icons adapt to your active resource pack.
- **Client-side & multiplayer-safe**
  - Does not send packets or automate gameplay.
  - Safe to use on Hypixel.
